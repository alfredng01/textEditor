# textEditor
A Vi Style Text Editor built in C++, based on my professors code.

#### Features
- File I/O
- Command/Normal Mode
- Text Coloring
- Undo/Redo

#### To Run:

1) Run `make`
2) Run `make clean` in order to get rid of all the object files
3) Run `./editor <filename>` and start typing!
